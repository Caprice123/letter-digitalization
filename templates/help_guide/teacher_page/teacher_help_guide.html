New

In this "New" tab, teacher can see all updates from the student who request an approval letter such as : Internship Letter, Active Student Letter, and Academic Transcript. Teacher can make a response by using accept or reject button in the response column.

Note:

If the teacher can see all records, then there will be a record can be seen but cannot be directly accepted or rejected because the record doesn't need the current role to make that response.

If teacher accept the request, then it will be forwarded to the operational and the status on student's dashboard will be approved by the teacher or HOP (1/2).

If teacher reject the request, then it will not be forwarded to the operational and the status on student's dashboard will be directly rejected.

All

In this "All" tab, teacher can see all request records which are already accepted or rejected and also the new incoming request.

Note:

If the teacher can see all records, then there will be a record can be seen but cannot be directly accepted or rejected because the record doesn't need the current role to make that response.

Accepted

In this "Accepted" tab, teacher can see all request records which are already accepted by him / her.

Note:

If the teacher can see all records, then there will be other records that are already accepted by him / her or other teacher accounts.

Rejected

In this "Rejected" tab, teacher can see all request records which is rejected by him / her.

Note:

If the teacher can see all records, then there will be other records that are already rejected by him / her or other teacher accounts.
