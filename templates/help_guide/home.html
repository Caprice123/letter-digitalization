Letter Digitalization

Welcome to the help guide for letter digitalization. This web app can make letter and send it automatically to specific teacher for confirmation. If the record is fully accepted, then the user will be emailed a letter with Dean signature. If the record is rejected, then the user will be updated via email. Every progress in the letter, the user will be updated via email also.
