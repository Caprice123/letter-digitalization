Create account

For creating new account, Student need to click an Create An Account located at the bottom right as shown below.

After clicking Create An Account button, Student must fill this form as shown below.

  • Name : Student name.

    • Please insert character only input, example: "Richie Eviendy".

  • NIM : Student NIM.

    • Please insert number only input, example: "2301919789".

  • Jurusan : Student Major.

  • Email : Student binus email.

    • Please insert email with extension of @binus.ac.id, example: "richie@binus.ac.id".

  • Password : The password of the student's account.

    • Please insert a strong password, example: "BiTco1nTHOTHEmoon".

  • Confirm Password : Retype password.

    • Please retype the password to make sure the password correct.

After fill the form, student need to click Register button located at the bottom right corner as shown below.

If the account is successfully created there is will be pop up as shown below.

Click back and sign in with the newly created account.

Student dashboard

After sign in, there will be a student dashboard as shown below.

Student can see all record that has been created and see its progress at this page.

Change password

If student want to change password, student must click the Student Name Menu located at top right corner be as shown below.

After click Change Password button, there is will be page as shown below.

Fill the form.

  • Current Password : Insert the last password of the student's account.

  • Create New Password : Insert the new password.

  • Confirm New Password : Retype the password.

After the form is filled up, click submit button.

If the password has been successfully changed, the user will be automatically log out and there will be flash message as shown below.
