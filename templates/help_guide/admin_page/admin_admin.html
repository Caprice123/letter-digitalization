Admin

This subpage of admin page handles all actions that admin can do with the all admin account except his / her account.

Add New Admin

For adding new admin, you need to click an add button located at the corner bottom right as shown below.

After clicking add button, there will be a popup window for the user to add new admin account as shown below.

In this popup window, there is some constraint for adding new admin as shown below.

  • Name : The name of the new admin.

    • Please insert character only input, example: "Admin".

  • Email : The binus email of the new admin

    • Please insert email with extension of @binus.ac.id, example: "admin.001@binus.ac.id".

After inserting forms, just click submit.

If success, there will be a popup that show the success message and the admin account will be saved and ready to be used.

If error, there will be a popup that show the error message and the admin account will not be saved.

Note:

When a new teacher account is created, the password of the account it the lowercase of the first name before space plus 123.
Example: Name: Marketing. Password: marketing123

Edit Admin

For editing admin, you need to click an edit button located at the left button of the actions column as shown below.

After clicking edit button, there will be a popup window for the user to edit other admin account as shown below.

In this popup, the user can edit other admin account. Please make sure the name is not the same. In this popup window, the constraint for editting the admin account is the same as the constraint for adding new admin account.

If already done editting the admin account, just click the submit button.

If success, there will be a popup that show the success message and the admin account will be saved.

If error, there will be a popup that show the error message and the admin account will not be saved.

Note:

When an admin account is successfully editted, the password of the account will stay the same.

Delete Admin

For deleting admin, you need to click an trash button located at the right button of the actions column as shown below.

After clicking delete button, there will be confirmation alert to confirm the actions. The confirmation alert will be as shown below.

If yes, then there will be a success popup the admin account will be deleted.

If no, then there will be no actions.
