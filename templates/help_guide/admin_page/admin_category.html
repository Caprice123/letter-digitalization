Category

This subpage of admin page handles all actions that admin can do

Add New Template

For adding new template, you need to click an add button located at the corner bottom right as shown below.

After clicking add button, there will be a popup window for the user to add new template as shown below.

In this popup, the user must insert new name for the category of the record. Please make sure the name is not the same. In this popup window, there is some constraint for adding the new category as shown below.

  • Category name: The subject of the letter.

    • Please insert character only input, example: "Surat Internship".

  • Category content: The content of the letter starting from the number until before signature.

    • If there is number letter, insert the following: {{'{{no}}/______/{{bulan terbit}}/{{year}}'}}. Example: {{'{{no}}/Dean.BASE/{{bulan terbit}}/{{year}}'}}.

    • If there is some dynamic data, please convert it into {{'{{___}}'}}. Example: Nama : {{'{{name}}'}}.

  • Available role: The role that is available for updating the record and someone with role who will get email everytime the progress of the record increases.

    • Just click whatever role for confirming the record and the role will be moved to accepted by section. Mind that the role must be in order.

  • Accepted By: The order which will be used for the record until it is accepted.

    • Click whenever you feel that the order is incorrect. Please keep in mind that the order is correct. Example:

      • Based on the image above, we can see that the order for the record before it is considered accepted is HOP → Operational.

After inserting all forms, just click submit and the category will be saved.

If success, there will be a popup that show the success message and the category will be visible to all student for making new record.

If error, there will be a popup that show the error message and the category will not be saved.

Note:

If there is number of letter, please insert it as following: {{'{{no}}/Dean.BASE/{{bulan terbit}}/{{year}}'}}.

If there is some date, please insert it as {{'{{date}} {{month}} {{year}}'}}

If the letter need the name of student, please insert it as {{'{{name}}'}}

If the letter need the nim of student, please insert it as {{'{{nim}}'}}

If the letter need the major of student, please insert it as {{'{{major}}'}}

If the letter need the subject, please insert it as {{'{{subject}}'}}

Edit Template

For editting template, you need to click an edit button located at response column as shown below.

After clicking edit button, there will be a popup window for the user to edit template as shown below.

In this popup, the user can edit the category of the record. Please make sure the name is not the same. In this popup window, the constraint for editting the category is the same as the constraint for adding new category.

If already done editting the category, just click the submit button and the category will be editted.

If success, there will be a popup that show the success message and the category will be visible to all student for making new record.

If error, there will be a popup that show the error message and the category will not be saved.

Note:

If there is number of letter, please insert it as following: {{'{{no}}/Dean.BASE/{{bulan terbit}}/{{year}}'}}.

If there is some date, please insert it as {{'{{date}} {{month}} {{year}}'}}

If the letter need the name of student, please insert it as {{'{{name}}'}}

If the letter need the nim of student, please insert it as {{'{{nim}}'}}

If the letter need the major of student, please insert it as {{'{{major}}'}}

If the letter need the subject, please insert it as {{'{{subject}}'}}

Disabling Template

For deleting the template, you need to click an trash button located at the second button at actions column as shown below.

After clicking the button, there will be a confirmation alert to confirm the actions. The confirmation alert will be as shown below.

If yes, then there will be a success popup the category will not be visible to the user or will be deleted and the category of the record will be shown with different color as shown below.

If no, then there will be no actions.

Enabling Template

For enabling the template, you need to click an refresh button located at the second button at actions column as shown below.

After clicking the button, there will be a confirmation alert to confirm the actions. The confirmation alert will be as shown below.

If yes, then there will be a success popup the category will be visible again to the user or will be deleted and the category of the record will be shown with different color as shown below.

If no, then there will be no actions.
