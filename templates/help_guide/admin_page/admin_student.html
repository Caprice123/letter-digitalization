Student

This subpage of admin page handles all actions that admin can do with the all student account.

Add New Student

For adding new student, you need to click an add button located at the corner bottom right as shown below.

After clicking add button, there will be a popup window for the user to add new student account as shown below.

In this popup window, there is some constraint for adding new student as shown below.

  • Name : The name of the student.

    • Please insert character only input, example: "Raymond".

  • NIM : The NIM of the student

    • Please insert integer only input, example: "2301919574".

  • Major : The major of the student

  • Email : The binus email of the student

    • Please insert character only input, example: "endra@binus.ac.id".

    • Please insert email with extension of @binus.ac.id, example: "endra@binus.ac.id".

    • Please insert a unique email

After inserting forms, just click submit.

If success, there will be a popup that show the success message and the student account is ready to be used.

If error, there will be a popup that show the error message and the student account is not created.

Note:

When a new student account is created, the password of the account it the lowercase of the first name before space plus 123.
Example: Name: Raymond W. Password: raymond123

Edit Student

For editing student, you need to click an edit button located at the left button of the actions column as shown below.

After clicking edit button, there will be a popup window for the user to edit student account as shown below.

In this popup, the user can edit the student account. In this popup window, the constraint for editting the student account is the same as the constraint for adding new student account.

If already done editting the student account, just click the submit button.

If success, there will be a popup that show the success message and the student account will be editted.

If error, there will be a popup that show the error message and the student account will not be editted.

Note:

When a student account is successfully editted, the password of the account will stay the same.

Delete student

For deleting student, you need to click an trash button located at the right button of the actions column as shown below.

After clicking delete button, there will be confirmation alert to confirm the actions. The confirmation alert will be as shown below.

If yes, then there will be a success popup the student account will be deleted.

If no, then there will be no actions.
