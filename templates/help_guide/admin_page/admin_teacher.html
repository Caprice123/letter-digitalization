Teacher

This subpage of admin page handles all actions that admin can do with the all teacher account.

Add New Teacher

For adding new teacher, you need to click an add button located at the corner bottom right as shown below.

After clicking add button, there will be a popup window for the user to add new teacher account as shown below.

In this popup window, there is some constraint for adding new teacher as shown below.

  • Name : The name of the teacher.

    • Please insert character only input, example: "Endra Oey".

  • Jabatan : The current position of the teacher

    • Please insert character only input, example: "HOP".

  • Jurusan : The major that the teacher is responsible.

    • Please insert full name of the major, example: Automotive and Robotics Engineering

    • Can leave blank if the teacher takes responsibility of all major.

  • Email : The binus email of the teacher

    • Please insert character only input, example: "endra@binus.ac.id".

    • Please insert email with extension of @binus.ac.id, example: "endra@binus.ac.id".

    • Please insert a unique email

  • Can see records : Whether the teacher can supervise or see all records according to their major and will receive the cc everytime the record is accepted or rejected.

After inserting forms, just click submit.

If success, there will be a popup that show the success message and the teacher account is ready to be used.

If error, there will be a popup that show the error message and the teacher account is not created.

Note:

When a new teacher account is created, the password of the account it the lowercase of the first name before space plus 123.
Example: Name: Endra Oey. Password: endra123

Edit Teacher

For editing teacher, you need to click an edit button located at the left button of the actions column as shown below.

After clicking edit button, there will be a popup window for the user to edit teacher account as shown below.

In this popup, the user can edit the teacher account. In this popup window, the constraint for editting the teacher account is the same as the constraint for adding new teacher account.

If already done editting the teacher account, just click the submit button.

If success, there will be a popup that show the success message and the teacher account will be editted.

If error, there will be a popup that show the error message and the teacher account will not be editted.

Note:

When a teacher account is successfully editted, the password of the account will stay the same.

Delete Teacher

For deleting teacher, you need to click an trash button located at the right button of the actions column as shown below.

After clicking delete button, there will be confirmation alert to confirm the actions. The confirmation alert will be as shown below.

If yes, then there will be a success popup the teacher account will be deleted.

If no, then there will be no actions.
